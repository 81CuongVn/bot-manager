# bot-manager
Manage Slash Commands with ease.

## Quick start
```
git clone 
https://github.com/GirlsFromPanema/bot-manager
npm i 
npm start
```
Enter your Bot token, then all of your Slash Commands will appear.
You can view, manage, delete them. Simple and Easy
